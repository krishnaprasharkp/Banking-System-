#  Banking System

A simple **Banking Management System** built in C++ that allows users to create accounts, deposit money, withdraw funds, check balances, and manage basic banking operations through a console-based interface.

## 🚀 Features

- Create new bank accounts
- Deposit money
- Withdraw money
- Check account balance
- View account details
- Menu-driven interface
- Basic transaction management

## 🛠️ Tech Stack

- C++
- Object-Oriented Programming (OOP)
- File Handling (if implemented)
- STL

## 📁 Project Structure

```bash
Banking-System/
│
├── banking.cpp

```

## ▶️ How to Run

```bash
g++ banking.cpp -o banking
./banking
```
